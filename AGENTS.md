# Sitar Hero — Agent Notes

## What This Is

A React + Vite frontend demo for sitar music exploration. Simple component-based architecture.

## Key Paths

| Path | Purpose |
|------|---------|
| `src/App.jsx` | Main app component |
| `src/main.jsx` | Vite entry point |
| `src/components/` | UI components |
| `vite.config.js` | Vite configuration |
| `package.json` | Dependencies and scripts |

## Dev Commands

```bash
npm install    # Install deps
npm run dev    # Dev server (port 5173)
npm run build  # Production build
```

## Safety Boundaries

- This is a frontend-only app. No backend, no secrets.
- Do not add API keys or external service integrations without review.
- Keep bundle size reasonable — avoid heavy dependencies.

## Status

Prototype. Safe to iterate on UI and content.
